Customer Insights – SaaS Platform UX

Interactive UX project showcasing research, wireframes, and prototypes

Project Overview

This project focuses on designing a user-friendly interface for a SaaS platform that helps businesses gather and analyze customer insights. The goal is to make complex data actionable and easy to understand.

UX Workflow Steps

  • User Research: Conducted surveys and interviews to understand user needs.
  • Persona Creation: Defined key user personas for targeted solutions.
  • Wireframes: Developed low-fidelity layouts for the main dashboard and analytics pages.
  • Interactive Prototype: Created clickable flows to demonstrate navigation and user interactions.
  • Usability Testing: Collected feedback and iterated designs for improved efficiency and clarity.

Impact & Key Learnings

The platform redesign improved clarity for decision-making and reduced navigation time by 30%. Learned how to balance data density with usability and visual hierarchy.
