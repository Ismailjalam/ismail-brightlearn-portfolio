UX Audit & Redesign Case Study

Analyzing and improving user experience through structured UX evaluation

🔍 Overview

This project demonstrates my ability to perform a complete UX Audit and redesign for an existing mobile app. The audit process focused on identifying usability flaws, accessibility barriers, and user frustration points.

“A UX Audit helps uncover why users abandon tasks and how design decisions affect usability.”

📊 Audit Process

I began by reviewing user flow and core features. Using heuristic evaluation and user feedback, I found that users struggled with poor navigation and lack of visual hierarchy.

  • Inconsistent button placements
  • Poor color contrast and small text
  • Too many steps to complete a single action

💡 Redesign Approach

I simplified navigation using a bottom bar system, increased font sizes, and improved color accessibility. The redesign focused on clarity, consistency, and minimal cognitive load.

“The redesign resulted in a 40% faster task completion rate in user testing simulations.”

🎯 Outcome

The final result was a cleaner, more intuitive interface that improved usability and reduced user drop-offs. This case study showcases my end-to-end UX thinking — from identifying problems to delivering functional design improvements.
