Advanced UX Flow – Interactive Prototype

Explore my detailed UX workflow and deliverables

1. User Research

Conducted surveys and interviews to understand user pain points and needs. Documented personas, empathy maps, and user scenarios to inform design decisions.

2. Wireframes

Created low-fidelity wireframes for web and mobile layouts to visualize structure and navigation. Iterated designs based on feedback.

3. Interactive Prototypes

Developed clickable prototypes demonstrating key workflows. Enabled stakeholders to experience the interface before development.

4. Testing & Feedback

Conducted usability testing sessions to identify issues. Incorporated actionable feedback into improved designs and documented the impact of changes.

5. Deliverables

Final deliverables include personas, wireframes, interactive prototype, user flow diagrams, and a mini case study highlighting challenges and solutions.
